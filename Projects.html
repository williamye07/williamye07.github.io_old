Weightlift Tracker

This was created using React for the frontend, Node.js for the backend and mysql database to store data. User's can input their training maxes and look at their progress by navigating the top menu. Also included is a training log to keep track of current reps and sets on a specific excercise. At the bottom is a bmi calculator. The project code can be viewed here https://github.com/williamye07/Lift-Tracker.

Twitch Dashboard

This project was created using React along with the twitch.tv api for data retrieval. The dashboard displays the games with the most viewers in real time. There are also options to view the most watched streamers of all games. It is also possible to click whichever stream you want to watch and it will allow you to watch it on twitch. The project code can be viewed here https://github.com/williamye07/Twitch-Dashboard.
